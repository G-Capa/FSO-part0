## Part 0 of the Full Stack Open Course by University of Helsinki.

This entire module is focused on the fundamentals of Web Apps, giving a introduction to how a web app handles requests and forms, how it uses files to control logic and styling and how to see all the events happening on a web-page in real time.
There is also a brief run through on the diferences between stndart web-pages and SPA, JavaScript libraries and what is AJAX.
