# 0.5-Single-page-app-diagram

Exercise number 0.5 from Full Stack Open Course by University of Helsinki

Diagram depicting the situation where the navigates to the the single-page app version of the notes app at https://studies.cs.helsinki.fi/exampleapp/spa.

```mermaid
graph TD
E{**BROWSER**} --> A["Navigate to Notes page"]
A --> F["Browser fetches HTML page"]
F --> G["Browser fetches JS file \(spa.js\)"]
G --> H["Browser fetches CSS file"]
G --> I["Browser sends GET request for /notes.json"]
I --> J["Server responds with notes data in JSON"]
J --> K["Browser stores data in `notes` array"]
K --> L["redrawNotes() renders notes on page"]

classDef browser fill:#fff7ed,color:#b45309
